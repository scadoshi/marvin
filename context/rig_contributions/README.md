# Rig Contributions

This directory tracks contributions to the [Rig framework](https://github.com/0xPlaygrounds/rig) discovered and submitted while building scotbot.

## Directory Structure

```
rig_contributions/
├── issues/              # Bug fixes and problem reports
│   ├── reported/        # Issues reported but not yet fixed
│   ├── pr_submitted/    # PRs submitted, awaiting review/merge
│   └── pr_approved/     # PRs merged and released
└── enhancements/        # Feature additions and improvements
    ├── reported/        # Enhancement ideas documented
    ├── pr_submitted/    # PRs submitted, awaiting review/merge
    └── pr_approved/     # PRs merged and released
```

## Workflow

1. **Discovery** → Document in `reported/`
2. **Submit PR** → Move to `pr_submitted/`
3. **Merged** → Move to `pr_approved/`
